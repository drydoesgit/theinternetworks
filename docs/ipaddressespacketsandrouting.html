IP Addresses, Packets and Routing

IP Addresses

So what are IP Addresses? The definition of a IP Address is: a unique string of numbers separated by periods that identifies each computer using the internet protocol to communicate over a network.

Example

The earlier example of a domain name was (www.icann.org), the IP address for this domain names web server is 192.0.32.7

Packets

What are packets? The definition of a packet is: a small segment of a larger message of data sent over computer networks, which are then recombined by the device that is receiving them to get the full message of data.

Example

Routing

What is routing? The definition of routing is: the process of selecting a path across one or more networks. This is how packets are sent across networks.
